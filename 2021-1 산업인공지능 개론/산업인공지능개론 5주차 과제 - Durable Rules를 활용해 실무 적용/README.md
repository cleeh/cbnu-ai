# 산업인공지능개론 5주차 과제
Durable Rules를 활용해 실무 적용

## 실무 요약
전자상거래 서비스가 구매자의 요청을 문제없이 처리했는지 확인할 필요가 있다.
확인할 사항들은 아래와 같다:
  - 상품
    - 이름
    - 가격
    - 개수
    - 총 가격
  - 고객
    - 이름
    - 적립금
    - 배송지 목록
  - 배송지
    - 일련번호
    - 이름
    - 지역번호
  - 주문
    - 지불 가격
    - 고객의 적립금 사용 무결성
